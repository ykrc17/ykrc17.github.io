Java中static块的用法

2014-10-22
Java

平时使用static关键字时候，动机很单纯，这个类里需要一个全局常量。这个常量如果是基本类型还好，如果是某个类呢。这时，可以用static块对全局常量进行初始化

static块

static块内的代码，在类被第一次加载、并且所有static常量初始化完成后，被执行。
static块只在类被加载时执行一次，之后不会再被执行。
多个static块会按照顺序执行，互相之间并无冲突。
